# Cab Booking System

## Overview

Cab Booking System is a console-based Java application developed to manage cab reservations efficiently. The system allows users to view available cabs, book rides, complete trips, cancel bookings, and manage ride details through a simple menu-driven interface.

## Features

* View Available Cabs
* Book a Cab
* Generate Unique Booking IDs
* Automatic Fare Calculation
* Complete Ride Management
* Cancel Bookings
* View All Bookings
* Cab Availability Tracking

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList Collections
* Scanner Class

## Project Structure

### Cab Class

Stores cab details such as:

* Cab ID
* Driver Name
* Cab Type
* Status

### RideData Class

Stores ride booking information:

* Booking ID
* User Name
* Pickup Location
* Drop Location
* Cab ID
* Fare
* Booking Status

### DataHandler Class

Manages:

* Cab Records
* Booking Records
* Data Retrieval Operations

### CabBookingManager Class

Handles:

* Cab Booking
* Fare Calculation
* Ride Completion
* Booking Cancellation

### CabBookingSystem Class

Provides the main menu-driven user interface.
